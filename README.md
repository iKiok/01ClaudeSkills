# Claude Skills Repository

Collection of custom skills for Claude Code.

## Skills

### polars-skill
Expert guidance for Polars dataframe manipulation in Python. Use when working with dataframes, data processing, ETL pipelines, or tabular data manipulation.

### scott.adams-reframe
Reframe problems using Scott Adams' philosophy from "Reframe Your Brain". Describe a problem, concern, or situation and get sardonic, pragmatic perspective shifts with:
- Before/After reframes
- Core theme identification
- Chemical angle (dopamine/serotonin/oxytocin mechanism)
- Actionable step
- Custom affirmation
- Philosophical closer

Inspired by [Reframe Your Brain](https://www.amazon.com/Reframe-Your-Brain-Interface-Happiness/dp/B0CGC8LSS1) by Scott Adams.

## Usage

1. Clone or download the skill folder you want
2. Place it in your `.claude/skills/` directory
3. Invoke with `/skill-name` or let Claude pick it up contextually
